## miniWebGameServer
* A mini game server by node.js.
* Design for web game, support HTTPS 
* storage using mysql.

--
### HowTo
* `git clone URI...`
* run `npm install` to install neccsary package
* run `node index.js` OR (exec `start.sh`(linux)  OR exec `start.bat`(windows) in ./tool/)
